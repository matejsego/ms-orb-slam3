# mono-and-stereo-orb-slam3
This repository is made for my final project: Localization of the AgileX Scout Mobile Robot Using Monocular and Stereo SLAM.

It is designed for Ubuntu 24.04 and ROS2 Jazzy environment.

Acknowledgments:

This project builds upon and integrates the following open-source repositories:

*   **[slam-wrapper-v2](https://github.com/BubbleRobotics/slam-wrapper-v2)**
*   **[nicla_vision_ros2](https://github.com/ADVRHumanoids/nicla_vision_ros2)**
*   **[nicla_vision_drivers](https://github.com/ADVRHumanoids/nicla_vision_drivers)**
